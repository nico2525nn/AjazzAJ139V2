# Ajazz AJ139 V2 コントロールソフト (非公式)

AJ139 V2 ワイヤレスマウスの設定・監視をデバイス上で（Webブラウザなしで）行うための非公式コントロールソフトウェアです。

![トレイ常駐 バッテリー監視](https://img.shields.io/badge/System%20Tray-Battery%20Indicator-brightgreen)

## 主な機能
- **通信設定へのフルアクセス**: Web版で動作するDPI、パフォーマンス設定、ライティングの全項目をローカルからネイティブに書き換え可能。
- **タスクトレイ常駐監視**: アプリの「✕」ボタンを押してもバックグラウンドで動作し続け、タスクバーのアイコン自体が直接「バッテリー残量(%)」を表示し続けます。
- **設定できるパラメーター**:
  - DPIレベル (最大6段階・オンザフライ切り替え可能)
  - レポートレート（ポーリングレート: 125 / 250 / 500 / 1000 Hz）
  - デバウンスタイム (Key Respond)
  - LOD (Lift-off Distance: 1mm / 2mm)
  - LEDオートスリープ時間およびライティングモード
- **多言語対応**: 自動的に「英語」と「日本語」のUIを切り替える機能を備えています。
- **デバッグログ**: USB通信間で送受信しているヘキサコード（16進数）生の通信ログを閲覧可能です。

## 動作環境

- **Python 3.8+**
- 必須パッケージ: `hidapi`, `pystray`, `Pillow`

## インストールとビルドの手順

1. **実行ファイル (.exe) を作りたい場合 (Windows)**:
   一番簡単な方法です。同梱されている `build.bat` をダブルクリックして実行するだけです。
   必要なライブラリのインストールから PyInstaller を使った単一ファイル(`OneFile`)へのビルドまでが自動で行われ、`dist`フォルダ内に `main.exe` が作成されます。

2. **Python スクリプトとして直接起動する場合**:
   ```bash
   pip install -r requirements.txt
   python main.py
   ```

## 使い方
1. PCにAjazz AJ139 V2 のUSBドングル、またはUSBケーブルを接続します。
2. アプリ（`main.exe`）を起動します。
3. **[最新状態を読み直す]** を押すとマウスから現在の設定一式がダウンロードされます。
4. **常駐機能について**: アプリの右上の「X」ボタンを押してもアプリは終了せず、タスクトレイ（右下）に最小化されます。システムトレイにある数字アイコンを右クリックし、「終了」を選ぶことで完全に終了します。
